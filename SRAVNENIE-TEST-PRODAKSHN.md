# Сравнение: тест (папка) и официальный сайт (REG.RU)

## Что с чем сравниваем

| | Адрес / место |
|---|---------------|
| **Тест** | Файлы в папке `optika-site-static` (ваш текущий код, локально или после push в GitHub) |
| **Оф. сайт** | https://optikadobryhcen.ru — то, что отдаёт хостинг (REG.RU / GitHub Pages) |

Отличия бывают, когда на сервере лежит **не та версия** файлов, что в папке, или мешает **кеш** браузера/CDN.

---

## Что должно быть на оф. сайте (как в тесте)

| Элемент | В тесте (папка) | Если на оф. сайте по-другому |
|--------|----------------------------------|------------------------------|
| **Версия в коде** | В `<head>` комментарий `<!-- ver:2026-03-11 -->` | Нет такого — залит старый index.html |
| **CSS** | Подключается `styles.css?v=2026031201` | Другое значение `?v=` или нет — подгружается старый стиль |
| **Герой** | Заголовок с ценой «2 990 ₽» целиком, подзаголовок «оправа + линзы с мультипокрытием + работа мастера» **без обрезки**, кнопка «Подобрать решение» **целиком** | Обрезанный текст («2 9», «мул мастера», «Подобрать реше») |
| **Ссылка «Мы в 2ГИС»** | В герое рядом с рейтингом, **светлая** (не синяя), ведёт на 2gis.ru/n_novgorod/firm/70000001032583529 | Синяя ссылка или другой URL |
| **Футер** | «Навигация», «Почему мы», «Добрая оптика», «Политика конфиденциальности» — **все слова целиком**, без обрезки | Обрезанные слова («Навигац», «Добрая о») |

| **Форма записи** | «3 варианта очков на выбор — от базового до премиального» в одну строку | Две строки или другой текст |
| **Надпись под формой** | «Отправляя заявку, вы соглашаетесь…» — хорошо читается | Почти не видна |

## Как быстро проверить: тест и оф. сайт совпадают или нет

1. Откройте **https://optikadobryhcen.ru**
2. Правой кнопкой → **«Просмотреть код страницы»** (View Page Source)
3. Поиск (**Ctrl+F** / Cmd+F) по строке: **`ver:2026-03-11`**
   - **Есть** — на оф. сайте отдаётся актуальный **index.html**. Дальше проверьте: в коде ссылка на стили должна быть `styles.css?v=2026031201`. Если в браузере открыть эту ссылку на CSS и поискать в файле `ver:2025-03-12` — значит подгружается нужный стиль.
   - **Нет** — на оф. сайте отдаётся **старый index.html**. Нужно залить из папки новый.

4. Визуально: герой (цена, подзаголовок, кнопки), футер (навигация), ссылка «Мы в 2ГИС» (цвет и URL) — как в таблице выше.

## Что сделать, чтобы оф. сайт совпадал с тестом

1. Залить из папки `optika-site-static` в **корень** сайта на REG.RU (и при необходимости обновить деплой из GitHub):
   - `index.html`, `styles.css`, `script.js`
   - все изображения, `send-form.php`, `politika-konfidencialnosti.html` и т.д.
2. **Перезаписывать** существующие файлы, не класть в подпапку.
3. После заливки открыть сайт **в режиме инкогнито** или с **Ctrl+Shift+R** (Cmd+Shift+R на Mac), чтобы сбросить кеш.

Итог: «тест» = эталон (папка / ветка main). Оф. сайт должен совпадать с ним после заливки и без старого кеша.
